## SCREEN RECORDING INSTRUCTIONS

### Setup
1. **Resolution**: 1920x1080 (Full HD)
2. **Frame Rate**: 60fps for smooth animations
3. **Browser**: Chrome (for best animation performance)
4. **Window Size**: Maximized, clear browser UI if possible
5. **Extensions**: Disable ad blockers and extensions
6. **Test Run**: Record 3-5 practice runs before final

### Recording Sequence

#### Pre-Record Checklist
- [ ] Clear browser cache
- [ ] Prepare test accounts (vendor + customer)
- [ ] Pre-fill demo data
- [ ] Test all animations work
- [ ] Close unnecessary tabs/applications
- [ ] Set "Do Not Disturb" mode
- [ ] Check audio input levels

#### Scene-by-Scene Recording

**Scene 1 - Homepage**
- Load homepage
- Let animations fully play (particles, statistics)
- Hold for 8 seconds

**Scene 2 - Features Rapid Fire**
- Navigate through each feature quickly
- Hover to trigger 3D tilt effects
- Show interactions clearly
- 4 seconds per feature maximum

**Scene 3 - Signup Flow**
- Start from homepage
- Click "Get Started Free"
- Fill form quickly (can speed up in editing)
- Select plan with clear click
- Submit and show success
- Dashboard loads with celebration

**Scene 4 - Split Screen Setup**
- Record vendor view separately
- Record customer view separately  
- Composite in editing

### Post-Production Tips
- Speed up slow sections (form filling can be 2x speed)
- Add smooth transitions between scenes
- Overlay text at exact timestamps
- Color grade for consistency
- Add motion blur if speeding up footage
- Stabilize any shaky mouse movements
